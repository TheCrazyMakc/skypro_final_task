Git

  • git add - команда используется для добавления файлов в индекс Git (промежуточную зону перед репозиторием). Это обязательный шаг перед тем, как совершить изменения в репозиторий с помощью git commit.
  • git commit -m - используется для сохранения изменений в локальном репозитории Git с сообщением, описывающим внесённые модификации.
  • git push origin main - Команда git push origin main используется для отправки локальных изменений в удаленный репозиторий.
  • git pull - используется для получения последних изменений из удаленного репозитория и их слияния с локальной версией.
